| Code | Meaning                                 |
| ---- | --------------------------------------- |
| 200  | OK (success)                            |
| 201  | Created (e.g., new user, post)          |
| 204  | No Content (success, nothing to return) |
| 400  | Bad Request (invalid input)             |
| 401  | Unauthorized (not logged in)            |
| 403  | Forbidden (no access)                   |
| 404  | Not Found                               |
| 409  | Conflict (e.g., duplicate email)        |
| 500  | Server Error                            |
